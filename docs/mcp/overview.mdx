Glot provides a [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) server that enables AI coding assistants to help you fix i18n issues directly in your editor.

## Setup

### Claude Code

Add glot to your MCP configuration:

```json mcp_config.json
{
  "mcpServers": {
    "glot": {
      "command": "npx",
      "args": ["glot", "serve"]
    }
  }
}
```

### Cursor

Configure glot in Cursor's MCP settings:

```json
{
  "mcp": {
    "servers": {
      "glot": {
        "command": "npx",
        "args": ["glot", "serve"]
      }
    }
  }
}
```

## What You Can Do

Once configured, your AI assistant can:

- **Find hardcoded text** - Scan your codebase for untranslated strings
- **Detect missing keys** - Identify translation keys used in code but not defined in locale files
- **Add translations** - Insert new translation keys across all your locale files

Example prompts you can use:

- "Check this file for hardcoded text"
- "Find all missing translation keys in my project"
- "Add translations for the new login form"

<Accordion title="Available MCP Tools">

### `get_config`

Returns the current glot configuration.

**Output:**

```json
{
  "fromFile": true,
  "config": {
    "primaryLocale": "en",
    "messagesDir": "./messages",
    "includes": ["src/app/[locale]", "src/components"],
    "ignores": [],
    "ignoreTestFiles": true,
    "ignoreTexts": [],
    "checkedAttributes": ["placeholder", "title", "alt", "aria-label"]
  }
}
```

### `get_locales`

Lists available locale files with key counts.

**Output:**

```json
{
  "messagesDir": "./messages",
  "primaryLocale": "en",
  "locales": [
    { "locale": "en", "filePath": "./messages/en.json", "keyCount": 156 },
    { "locale": "es", "filePath": "./messages/es.json", "keyCount": 152 }
  ]
}
```

### `scan_overview`

Returns statistics of all i18n issues in the project.

**Output:**

```json
{
  "hardcoded": {
    "totalCount": 12,
    "fileCount": 5
  },
  "primaryMissing": {
    "totalCount": 4
  },
  "replicaLag": {
    "totalCount": 8,
    "affectedLocales": ["es", "fr"]
  }
}
```

### `scan_hardcoded`

Returns detailed hardcoded text issues with pagination support.

**Input:**

| Parameter | Type   | Description                     |
| --------- | ------ | ------------------------------- |
| `limit`   | number | Results per page (default: 100) |
| `offset`  | number | Offset for pagination           |

**Output:**

```json
{
  "totalCount": 12,
  "totalFileCount": 5,
  "items": [
    {
      "filePath": "./src/components/Button.tsx",
      "line": 5,
      "col": 22,
      "text": "Submit",
      "sourceLine": "    return <button>Submit</button>;"
    }
  ],
  "pagination": {
    "offset": 0,
    "limit": 100,
    "hasMore": false
  }
}
```

### `scan_primary_missing`

Lists translation keys missing from the primary locale.

**Output:**

```json
{
  "totalCount": 4,
  "items": [
    {
      "key": "common.submit",
      "filePath": "./src/components/Button.tsx",
      "line": 15
    }
  ],
  "pagination": {
    "offset": 0,
    "limit": 100,
    "hasMore": false
  }
}
```

### `scan_replica_lag`

Lists keys that exist in the primary locale but are missing from other locales.

**Output:**

```json
{
  "totalCount": 8,
  "items": [
    {
      "key": "common.newFeature",
      "value": "New Feature",
      "existsIn": "en",
      "missingIn": ["es", "fr"]
    }
  ],
  "pagination": {
    "offset": 0,
    "limit": 100,
    "hasMore": false
  }
}
```

### `add_translations`

Adds new translation keys to one or more locale files.

**Input:**

```json
{
  "translations": [
    {
      "locale": "en",
      "keys": {
        "common.submit": "Submit",
        "common.cancel": "Cancel"
      }
    },
    {
      "locale": "es",
      "keys": {
        "common.submit": "Enviar",
        "common.cancel": "Cancelar"
      }
    }
  ]
}
```

**Output:**

```json
{
  "success": true,
  "results": [
    {
      "locale": "en",
      "success": true,
      "filePath": "./messages/en.json",
      "addedCount": 2,
      "updatedCount": 0
    }
  ],
  "summary": {
    "totalLocales": 2,
    "successfulLocales": 2,
    "failedLocales": 0,
    "totalKeysAdded": 4,
    "totalKeysUpdated": 0
  }
}
```
